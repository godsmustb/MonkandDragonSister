# The Monk & The Dragon Sister

A two-player split-screen brawler set in a jade garden — co-op beat-em-up where a kung-fu monk and a shape-shifting dragon sister fight waves of elemental demons across five escalating boss encounters.

## HOW TO PLAY

Double-click **play.bat** (needs Python installed; or run any static server and open `index.html` via `http://localhost`).

## CONTROLS

### Player 1 — The Monk (left screen)
| Key | Action |
|-----|--------|
| W A S D | Move |
| Space / I | Attack (3-hit combo) |
| J | Chi Shield (protects both players, 2.5 s) |
| K | Dodge (i-frames, rolls back) |
| L | Healing Pulse (AoE heal) |
| Q / E | Camera orbit left / right |
| F | Lock-on toggle (targets nearest demon) |

### Player 2 — Dragon Sister (right screen)
| Key | Action |
|-----|--------|
| Arrow Keys | Move |
| Enter / Numpad 8 | Attack |
| Numpad 4 | Transform (cycle dragon forms) |
| Numpad 5 | Dodge |
| Numpad 6 | Special ability |
| Numpad 7 / 9 | Camera orbit left / right |
| Numpad 0 | Lock-on toggle |

### Shared
| Key | Action |
|-----|--------|
| M | Mute / unmute audio |
| Esc | Pause / resume |

## ELEMENTAL COUNTER RING

> **Water > Fire > Ice > Poison > Water** — and round again.

Hitting a demon with its counter element deals **2x damage**; hitting with the element it counters deals **0.5x damage**. Watch the advantage chip on your HUD to know your match-up before committing.

## QUEST STRUCTURE

Five waves guard the garden:

1. **Wave I** — Shadowlings (neutral)
2. **Wave II** — Frost Imps (ice)
3. **Wave III** — Tide Wraiths (water)
4. **Wave IV** — Venom Oni mini-boss (poison)
5. **Wave V** — Inferno Demon Lord final boss (fire)

Defeat all five to cleanse the garden and unlock the true ending.

## REQUIREMENTS

- Modern browser with **WebGL 2** support (Chrome 80+, Firefox 75+, Edge 80+)
- **Internet connection on first load** — Three.js and addons are loaded from the jsDelivr CDN
